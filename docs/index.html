Overview

DustDB is designed to be a simple and efficient key-value store. It supports basic operations like GET, SET, INCR, DECR, and PING, making it suitable for various high-performance applications. Crafted with efficiency in mind, DustDB outperforms traditional caching solutions by utilizing advanced memory management techniques and OS level optimizations.

Usage

To start using DustDB, run the following command:

make
Copied!

Once the server is running, you can interact with DustDB through its command interface. By default, the server listens on port 6379, but this can be configured in the config.js file.

Commands

The following commands are available in DustDB:

  • GET key

    Retrieve the value associated with the specified key.

    dustdb> GET user:profile:1
    Copied!

  • SET key value

    Set the value for the specified key.

    dustdb> SET user:profile:1 '{"name":"John","age":30}'
    Copied!

  • INCR key

    Increment the integer value of the specified key by one.

    dustdb> INCR pageviews:homepage
    Copied!

  • DECR key

    Decrement the integer value of the specified key by one.

    dustdb> DECR available:tickets
    Copied!

  • PING

    Check the connection to the database.

    dustdb> PING
    Copied!
